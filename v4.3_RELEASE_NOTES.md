## ✨ What's New in v4.3

### 🎨 Complete Receipt Customization System
**User Control Over Receipt Formatting**

- **ALL Size Options Available**: 1x, 1.2x, 1.25x, 1.5x, 2x
- **Bold Toggle** for every field
- **Customizable Fields**:
  - Business information (name, address, phone)
  - Ticket ID
  - Vehicle details (number, type)
  - Travel details (header, from, to locations)
  - Total amount

**How to Use:**
1. Go to Settings → Receipt Customization
2. Select size and bold for each field
3. Tap Save
4. Changes apply immediately to all receipts

---

### 🔌 USB Printer Support (Android)
**Connect Thermal Printers via USB OTG**

- **Automatic Device Detection**
- **Supported Brands**:
  - SEWOO thermal printers
  - Star Micronics
  - Epson thermal printers
  - GOOJPRT
  - XPrinter
  - Generic USB thermal printers

**How to Use:**
1. Connect USB OTG cable to your phone
2. Connect thermal printer to OTG cable
3. Go to Settings → Printer Settings
4. Select "USB" connection type
5. Tap "Scan" button
6. Tap "Connect" on your printer
7. Ready to print!

**Features:**
- Auto-reconnect on app start
- Same print quality as Bluetooth
- No pairing required
- Works with most ESC/POS thermal printers
